资源加速器

轻松访问 JSDelivr、GitHub 和 Docker 资源，无需任何限制

支持的链接类型(可直接拼接URL)：

JSDelivr NPM 包: cdn.jsdelivr.net/npm/package@version/file
JSDelivr GitHub 文件: cdn.jsdelivr.net/gh/user/repo@version/file
JSDelivr WordPress 插件: cdn.jsdelivr.net/wp/plugins/plugin-name/version/file
JSDelivr 组合链接: cdn.jsdelivr.net/combine/npm/package1,npm/package2
GitHub Release/Archive: github.com/user/repo/releases/download/v1.0.0/file
GitHub Raw 文件: raw.githubusercontent.com/user/repo/branch/file
Docker 镜像: registry-1.docker.io/library/image:tag
