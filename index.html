资源加速器

轻松访问 JSDelivr、GitHub 和 Docker 资源，无需任何限制

支持的链接类型（可直接拼接 URL）

JSDelivr NPM
cdn.jsdelivr.net/npm/package@version/file
JSDelivr GitHub
cdn.jsdelivr.net/gh/user/repo@version/file
JSDelivr WordPress
cdn.jsdelivr.net/wp/plugins/plugin/ver/file
JSDelivr 组合
cdn.jsdelivr.net/combine/npm/pkg1,npm/pkg2
GitHub Release
github.com/user/repo/releases/download/v1/file
GitHub Raw
raw.githubusercontent.com/user/repo/branch/file
Docker 镜像
registry-1.docker.io/library/image:tag
Docker 镜像名
nginx:latest、ubuntu:22.04、user/repo:tag
